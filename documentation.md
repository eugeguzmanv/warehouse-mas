RobotAgentRole: The active worker in the system.State:RETRIEVING: Searching for a box to pick up.STACKING: Carrying a box to a shelf.Capabilities:Perception: Can see boxes and shelves on the entire grid (Simplified "Global Knowledge").Movement: Can move in 4 cardinal directions (North, South, East, West).Collision Avoidance: Checks if a target cell contains another robot before moving. BoxRole: Passive entity.Behavior: Sits on the grid until picked up. Once picked up, it is removed from the grid representation.C. ShelfRole: Target destination.Behavior: Keeps track of stack_height. Max capacity is 5.Visualization: Changes color from Grey (Empty) to Black (Full) and displays current count.2. Environment (model.py)Grid: A 20x20 MultiGrid. This allows multiple agents to theoretically exist in one cell (e.g., a Robot on top of a Shelf), but logic prevents Robot-Robot collision.Scheduler: RandomActivation. In every step, agents are activated in a random order to prevent deterministic movement artifacts.Data Collection: Tracks "Total Movements" (efficiency) and "Boxes Stacked" (progress) for real-time graphing.3.
